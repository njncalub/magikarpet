# magikarpet
Carpet inventory system using [MongoDB](https://www.mongodb.com/download-center) and [Polygon 3](https://www.j-raedler.de/projects/polygon/).

### Requirements
* Python >=3.5.x
* MongoDB >=3.6.x

### Recommended
I recommend installing the following version managers to make your life easier:
* [pyenv](https://github.com/pyenv/pyenv)
